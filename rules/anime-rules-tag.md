# 番剧 TAG 规则

编写和维护番剧信息的 TAG 字段时，须遵循以下规则。


### TAG 字段规则

#### 定义

TAG 字段用于以**简短词语**描述番剧的核心剧情特征与题材属性，便于快速了解作品定位与风格。

#### 搜集原则

1. **参考数据源 TAG**：优先参考 Bangumi 和 anime-planet 上的用户/官方标签。
2. **剧情校验**：必须结合故事简介、PV、官方网站等**多方剧情信息**进行校验，剔除与剧情不符的标签，补充遗漏的关键标签。
3. **归纳精简**：最终输出的 TAG 应为**精简后的中文词语**，避免过长或过于冷门的表达。

#### 输出格式

- 多个 TAG 之间用中文顿号「、」分隔
- 每个 TAG 控制在 2~6 个汉字，特殊情况（如英文专有名词）除外
- 如无合适 TAG，标记为 `-`
- **TAG 数量要求**：每部番剧至少 **2** 个 TAG，最多 **10** 个 TAG，以能够准确描述番剧类型为准，不必刻意精简或填满限额
- **TAG 排序规则**：按照先主后次的顺序排列。例如「科幻」「奇幻」等描述核心类型的 TAG 通常靠前；「后传」「外传」「Q版」「芳文社」等辅助性、衍生性 TAG 放在末尾

#### 特殊 TAG 类别参考及解释

- 萝卜：Robot，即“机甲”。
- 贴贴：女孩子之间有超越友谊的亲密互动，但程度不及百合，也没有明确定性为恋爱。
- 涩涩：即“色色”，有色情向内容或擦边内容。
- 福利：有擦边内容，但程度不及“涩涩”。
- 狗粮：两个人亲密无间，公开秀恩爱、发糖，非常惹人羡慕。
- 给里给气：即“Gay里Gay气”，像Gay但又不是Gay。
- 抽象：作品表现手法或剧情走向超出常规逻辑，难以用通常标准理解。
- 胃疼：剧情充满纠结、误会、虐心元素，让观众感到焦虑或心理不适。
- 幼驯染：即”青梅竹马”之意。
- 整活：作品中有刻意制造的荒诞、搞笑、出其不意的情节或桥段。
- 玩梗：作品中大量引用、致敬其他作品、文化现象或网络迷因。
- 装逼：主角以压倒性实力或气场展示能力，带有强烈的优越感。
- 龙傲天：男主角开局即拥有超强实力，一路碾压对手的无敌流作品。
- 凤傲天：女主角开局即拥有超强实力，一路碾压对手的无敌流作品。
- 开挂：主角拥有远超常人的能力或运气，以不合理的方式解决问题。
- 养女儿：剧情核心围绕主角养育、照顾年幼女性角色展开。
- 芳文社：原作或出品与芳文社（Manga Time Kirara 系）相关，多为轻百合、萌系日常风格。
- 退队流：主角被队伍驱逐后，以更强姿态回归或独立发展的情节模式。
- 恶役千金：主角身为反派大小姐，走向与原作反派不同命运的题材。
- 转生恶役：转生到游戏/小说世界成为反派角色，努力回避 Bad End 的题材。
- 雌小鬼：年幼女性对年长者表现出嚣张、挑衅态度的角色类型。
- 假小子：行为举止、外表偏男性化的女性角色。
- 人外：非人类角色，如妖怪、兽人、机器人、神明等。
- 沙雕：作品风格极度搞笑、荒诞，以无厘头幽默为主要卖点。
- 无厘头：缺乏逻辑铺垫、出其不意的搞笑风格。
- 慢生活：剧情核心为逃离快节奏争斗，享受悠闲平静的日常生活。
- 王道：遵循传统英雄叙事，强调正义、友情、努力的经典热血路线。
- Q版：角色以二头身/简化萌化造型出现。
- 反差：角色性格、身份、行为与外表形成强烈对比。
- 逆天改命：主角反抗命运安排，试图改变既定结局。
- 特摄衍生：源自特摄片（如假面骑士、奥特曼等）的动画衍生作品。
- 猎奇：内容怪异、反常，以挑战观众心理承受力为特点。
- 黑色幽默：以荒诞、戏谑的方式处理严肃或悲剧性主题。
- 荒诞：脱离现实逻辑的反讽风格，常与存在主义主题相关。

#### 注意事项

- TAG 应优先聚焦于**剧情内容本身**，而非制作信息（如「新房昭之」「京都动画」不应作为 TAG），但有一个例外：芳文社
- 避免过于笼统的标签（如「动画」「日本」）
- 续作番剧需结合**当季剧情**归纳 TAG，而非直接沿用上季度 TAG，除非剧情概况完全一致（例如一季内容分割成两部分播放）
