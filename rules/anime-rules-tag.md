# 番剧 TAG 规则

编写和维护番剧信息的 TAG 字段时，须遵循以下规则。


### TAG 字段规则

#### 定义

TAG 字段用于以**简短词语**描述番剧的核心剧情特征与题材属性，便于快速了解作品定位与风格。

#### 搜集原则

1. **参考数据源 TAG**：优先参考 Bangumi 和 anime-planet 上的用户/官方标签。
2. **剧情校验**：必须结合故事简介、PV、官方网站等**多方剧情信息**进行校验，剔除与剧情不符的标签，补充遗漏的关键标签。
3. **归纳精简**：最终输出的 TAG 应为**精简后的中文词语**，避免过长或过于冷门的表达。

#### 输出格式

- 多个 TAG 之间用中文顿号「、」分隔
- 每个 TAG 控制在 2~6 个汉字，特殊情况（如英文专有名词）除外
- 如无合适 TAG，标记为 `-`
- TAG 数量建议控制在 **3~8 个**

#### 常见 TAG 类别参考

| 类别 | 示例 |
|------|------|
| 题材类型 | 校园、职场、运动、音乐、美食、旅行、战争、推理、悬疑 |
| 核心元素 | 魔法、异世界、转生、时间穿越、机甲、吸血鬼、僵尸、恋爱、战斗、打斗、战争 |
| 情感基调 | 治愈、催泪、搞笑、热血、致郁、恐怖、日常、励志 |
| 受众指向 | 乙女向、后宫、百合、基腐、子供向、女性向 |
| 风格特征 | 赛博朋克、废土、蒸汽朋克、萌系、写实、意识流 |

#### 特殊 TAG 类别参考及解释

- 萝卜：Robot，即“机甲”。
- 贴贴：女孩子之间有超越友谊的亲密互动，但程度不及百合，也没有明确定性为恋爱。
- 涩涩：即“色色”，有色情向内容或擦边内容。
- 狗粮：两个人亲密无间，公开秀恩爱、发糖，非常惹人羡慕。

#### 注意事项

- TAG 应优先聚焦于**剧情内容本身**，而非制作信息（如「新房昭之」「京都动画」不应作为 TAG）
- 避免过于笼统的标签（如「动画」「日本」）
- 续作番剧需结合**当季剧情**归纳 TAG，而非直接沿用上季度 TAG，除非剧情概况完全一致（例如一季内容分割成两部分播放）
