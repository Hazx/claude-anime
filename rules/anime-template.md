# 番剧信息文件输出格式（模板）

每部番剧的 Markdown 文件采用以下格式（纯文本列表，便于复制粘贴）：

```markdown
# {译名}

- 译名：{译名}
- 别名：
  - {别名1}
  - {别名2}
- 日文原名：{日文原名}
- 英文译名：{英文译名}
- 番剧类型：{类型}
- 官网网址：{网址}
- 开播时间：{4月1日 00:05 / 4月1日 / 2026年春季}
- 集数：{共12集 / 共2季 播1季 / 季番}
- 全球独占：{平台名称 或 -}
- 泡面番：{是 或 -}
- 多版本：{是 或 -}
- TAG：{标签1、标签2、标签3}

## 故事简介

{简介内容}

## 主要 CAST

- {角色名}：{声优名}
- {角色名}：{声优名}

## 主要 STAFF

### 完整 STAFF

- {职位}：{姓名}
- {职位}：{姓名}

### 精选 STAFF

- {职位}：{姓名}
- {职位}：{姓名}
```

> **格式规则**已单独抽离至 [`anime-rules-format.md`](anime-rules-format.md)，在编写和维护番剧信息内容时须同时参考该文件。
>
> **别名填写说明**：若番剧无别名，`- 别名：` 后直接写 `-`，即 `- 别名：-`；若有别名，则在下一行以 `- {别名}` 的形式逐项列出。
